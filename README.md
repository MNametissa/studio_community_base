# Studio Community Base

Base components for running Odoo Studio on Community Edition.

## Description

This module provides stub implementations of `web_enterprise` components required by Odoo Studio to run on Odoo Community Edition.

## Features

- **HomeMenu Service** - Provides the `home_menu` service interface
- **ColorScheme Service** - Provides the `color_scheme` service interface
- **HomeMenu Component** - OWL component for the home menu display
- **EnterpriseNavBar Component** - Navigation bar component compatible with Studio
- **ListRenderer Extensions** - Desktop list renderer enhancements

## Requirements

- Odoo 16.0 Community Edition
- Python 3.8+

## Installation

1. Place this module in your Odoo addons path
2. Update the apps list in Odoo
3. Install "Studio Community Base" from the Apps menu

## Dependencies

- `web`
- `base`
